# 洋山

主页: http://192.168.203.200:8080/

## 开发

```zsh
git clone git@codehub.devcloud.huaweicloud.com:Demo29121/yangshan.git

cd yangshan
yarn
yarn dev
```

启动 `webpack` 服务

```zsh
yarn dev
```

**之后**，在你的浏览器中打开 http://127.0.0.1:8080/ ，开始你的表演。
