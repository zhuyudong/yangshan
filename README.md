## Development

```zsh
git clone https://github.com/zhuyudong/yangshan.git

cd yangshan
yarn
yarn dev
```

Start `webpack` server

```zsh
yarn dev
```

**Next**, your show time.

Open url http://127.0.0.1:8080/ in browser.
